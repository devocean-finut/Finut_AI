## Git Convention

### Commit Rule
```
__Action__ FileName__ : __Description__
```
- Action
  - added: 파일 추가
  - update: 파일 수정
  - remove: 파일 삭제

### Issue/PR Rule
```
[TYPE] TITLE
```
- TYPE
  - DEV: 일반적인 개발 사항
  - FIX: 기존에 개발된 내용 수정
  - DOC: README등 문서 관령 사항

### Branch Rule
```
TYPE/BRANCH_NAME
```
- TYPE
  - dev: 일반적인 개발 사항
  - fix: 기존에 개발된 내용 수정
  - doc: README등 문서 관령 사항
